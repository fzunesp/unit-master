# 📐 Unit Master

A premium, self-contained unit conversion tool with live currency exchange. Built as a single HTML file — zero dependencies, zero installs.

## ✨ Features

- **32 conversions** across 8 categories: Length, Weight, Temp & Speed, Volume, Area/Time/Data, Currency, Cooking, Fuel & Pressure
- **20 live currencies** with real-time exchange rates via the fawazahmed0 Currency API
- **Snap to share** — capture any card as a PNG image (clipboard or download)
- **Color-coded tabs** with smooth animated transitions
- **Keyboard shortcuts** — press `1–8` to switch tabs instantly
- **Smart number formatting** — adaptive precision, no trailing zeros
- **Mobile ready** — horizontal-scroll tabs, iOS zoom prevention, responsive layout
- **Premium glassmorphism UI** with an animated aura background

## 🚀 Deployment

This is a static site — served as a single `index.html` file. No build step required.

### Coolify (Static Site)
1. Connect this GitHub repo to Coolify
2. Set resource type to **Static Site**
3. Set your domain
4. Deploy — SSL is auto-provisioned via Let's Encrypt
